Lecturas

¿Qué es GITHUB y para qué se utiliza?
GITHUB es un sitio web, es la plataforma donde se almacenan los
proyectos que desarrollamos y subimos a nuestro repositorio y se utiliza como una red social diseñada para hacer más fácil el
trabajo de los desarrolladores por medio de trabajos colaborativos a nivel mundial.

¿Cómo usar GITHUB?
Para utilizar esta plataforma debemos crear una cuenta de GITHUB y descargar GITHUB
para escritorio. Posteriormente, creamos un repositorio en nuestra cuenta y daremos conexión a este con el GITHUB QUE DESCARGAMOS
¿Qé es GIT?
GIT es el software, la herramienta que utilizaremos para subir nuestros proyectos a GITHUB.
Este software se encarga de hacer seguimiento a todos los cambios que realicemos en nuestros archivos. GIT los va registrando
a medida que se desarrollen y deja un registro de cada vez que se guarden y se suban estas modificaciones a GITHUB.
