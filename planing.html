¿Qué es GITHUB y para qué se utiliza?
GITHUB es un sistema de gestión de proyectos y control de versiones de código, que a su vez, es también una plataforma de red social diseñada para hacer más fácil el trabajo de los desarrolladores por medio de trabajos colaborativos a nivel mundial.
