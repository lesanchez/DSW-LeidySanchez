Lecturas

¿Qué es GITHUB y para qué se utiliza?
GITHUB es un sitio web, es la plataforma donde se almacenan los
proyectos que desarrollamos y subimos a nuestro repositorio y se utiliza como una red social diseñada para hacer más fácil el
trabajo de los desarrolladores por medio de trabajos colaborativos a nivel mundial.

¿Cómo usar GITHUB?
Para utilizar esta plataforma debemos descargar GIT para escritorio y registrarse en una cuenta para GITHUB. Una vez creada
la cuenta, procedemos a crear un repositorio y daremos conexión a este con el GITHUB que descargamos anteriormente.

¿Qé es GIT?
GIT es el software, la herramienta que utilizaremos para subir nuestros proyectos a GITHUB. Este software se encarga de hacer
seguimiento a todos los cambios que realicemos en nuestros archivos. GIT los guarda bajo nuestra orden a medida que se desarrollen
y deja un registro de cada vez que se guarden y se suban estas modificaciones a GITHUB.

¿Cómo crear una cuenta y un repositorio en GITHUB?
Lo primero que necesitamos hacer, es crear ua cuenta para ususario gratuita ingresando a github.com y siguiendo los pasos que allí
nos solicitan. finalizado este proceso, seguiremos con la creación de nuestro repositorio para después sincronizarlo con nuestro GIT
de escritorio que fue descargado con anterioridad.
